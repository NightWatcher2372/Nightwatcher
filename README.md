# Solana Message Storage Program

## Descripción

Este proyecto es un programa simple desarrollado para la blockchain de Solana utilizando Rust.  
El objetivo del programa es permitir que un usuario envíe un mensaje y que este sea almacenado dentro de una cuenta en la blockchain.

Este ejemplo demuestra conceptos básicos del desarrollo en Solana como:

- Programas nativos en Rust
- Manejo de cuentas
- Serialización de datos con Borsh
- Envío de instrucciones desde un cliente

---

## Tecnologías utilizadas

- Rust
- Solana Program Library
- Borsh (Binary Object Representation Serializer for Hashing)
- @solana/web3.js
